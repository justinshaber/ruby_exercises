PROBLEM
  OVERVIEW - Given two strings of DNA, calculate the hamming distance.
    Calculate the number of differences between two strings.

  INPUT: two strings
  OUTPUT: integer

EXAMPLES/RULES
  RULES
    EXPLICIT
      If given two strings of different lengths, use the length of the shorter
    IMPLICIT
      empty strings return 0

  METHODS
    #initialize
      - accepts one string argument
      - initializes a instance variable to the passed string
    #hamming_distance
      - accepts a string argument
      - calculates and returns the hamming distance

DATA STRUCTURES
  Arrays

ALGORITHM
  #initialize(dna)
    INIT instance var dna
  #hamming_distance(other_dna)
    FIND shorter
    INIT mutations
    ITERATE over each_char.with_index
      INCREMENT counter if char != longer
